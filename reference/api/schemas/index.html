    API Schemas¶

    Pydantic models for request/response validation and serialization.

    Projects¶

    projects ¶

    Pydantic schemas for project-related requests and responses.

    CreateProjectRequest ¶

    Bases: BaseModel

    Request body for creating a new labelling project.

    AddLabelerRequest ¶

    Bases: BaseModel

    Request body for adding a labeler to a project.

    UpdateProjectRequest ¶

    Bases: BaseModel

    Request body for updating a project's mutable fields.

    LabelerInfo ¶

    Bases: BaseModel

    Summary information for a labeler assigned to a project.

    ProjectSummary ¶

    Bases: BaseModel

    Lightweight project representation for list endpoints.

    ProjectDetail ¶

    Bases: BaseModel

    Full project representation with resolved labeler details.

    PaginatedProjectSummaryResponse ¶

    Bases: PaginatedResponse[ProjectSummary]

    Paginated response containing a list of project summaries.

    Images¶

    images ¶

    Pydantic schemas for image-related requests and responses.

    AddImageRequest ¶

    Bases: BaseModel

    Request body for registering an existing storage object as a project image.

    ImageResponse ¶

    Bases: BaseModel

    Response body for an image resource.

    SubmitLabelRequest ¶

    Bases: BaseModel

    Request body for submitting a label for an assigned image.

    LabelRecordResponse ¶

    Bases: BaseModel

    Response body for a label record.

    SubmitLabelResponse ¶

    Bases: BaseModel

    Response body for a submitted label, including labeler ranking.

    ImportImagesRequest ¶

    Bases: BaseModel

    Request body for starting a bulk image import from storage.

    ImportJobResponse ¶

    Bases: BaseModel

    Response body for an import job with its current status.

    Assignments¶

    assignments ¶

    Pydantic schemas for image assignment requests and responses.

    CreateAssignmentRequest ¶

    Bases: BaseModel

    Request body for creating a new image assignment.

    AssignmentResponse ¶

    Bases: BaseModel

    Response body for a created image assignment with presigned URL.

    Pagination¶

    pagination ¶

    Generic pagination schema for paginated API responses.

    PaginatedResponse ¶

    Bases: BaseModel, Generic[T]

    Generic paginated response wrapper with items, total count, and offset.

    Stats¶

    stats ¶

    Pydantic schemas for project statistics responses.

    LabelerClassCountSchema ¶

    Bases: BaseModel

    Per-labeler label class counts within a project.

    DailyCountSchema ¶

    Bases: BaseModel

    Label counts by class for a single day.

    LabelerDailyActivitySchema ¶

    Bases: BaseModel

    Daily labelling activity breakdown for a single labeler.

    DailyLabelerTotalSchema ¶

    Bases: BaseModel

    Total label count for a single labeler on a single day.

    DailyTotalSchema ¶

    Bases: BaseModel

    Aggregate daily totals with per-labeler breakdowns.

    ProjectStatsResponse ¶

    Bases: BaseModel

    Complete labelling statistics for a project.

    Tokens¶

    tokens ¶

    Pydantic schemas for authentication requests and responses.

    LoginRequest ¶

    Bases: BaseModel

    Request body for user sign-in by username.

    Claim ¶

    Bases: BaseModel

    Response body containing the authenticated user's identity and role.

    Errors¶

    errors ¶

    Pydantic schemas for structured API error responses.

    ErrorBody ¶

    Bases: BaseModel

    Inner error object containing code, message, and details.

    ErrorResponse ¶

    Bases: BaseModel

    Wrapper schema for API error responses.
